# Grid

| Options | Type | Default |
| ----------- | ----------- | ----------- |
| ColumnNumber | Number | 12 |
| FullHeight | Boolean | False |
| FullWidth | Boolean | False |
| GridColor | String | #00000015 |
| GridGap | Number | 12 |
| GridHorizontal | Boolean | False |
| GridUnity | String | px |
| GridZIndex | Number | -1 |


# Usage
```js
import projectGrid from 'grid'

const grid = new grid({ 
  option: a,
  option: b
})

projectGrid.createGrid()
```
